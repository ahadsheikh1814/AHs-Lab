## Bun Package Manager

All commands in this documentation use **Bun** as the package manager.

If you haven't installed Bun yet, run:

```bash
curl -fsSL https://bun.sh/install | bash
```

## Next.js

This project uses Next.js for building modern React applications.

## Tailwind CSS

All styling is done with Tailwind CSS.

To install Tailwind CSS, run:

```bash
npm install tailwindcss @tailwindcss/postcss postcss
```

Then configure PostCSS plugins in postcss.config.mjs:

```js
const config = {
  plugins: {
    "@tailwindcss/postcss": {},
  },
};
export default config;
```

Import Tailwind CSS in your global stylesheet **globals.css**:

```css
@import "tailwindcss";
```

Now you can use Tailwind classes in your components, for example:

```tsx
export default function Home() {
  return <h1 className="text-3xl font-bold underline">Hello world!</h1>;
}
```
